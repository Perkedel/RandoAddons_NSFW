# Wahaha Dice "✌️ Two"

Spin off of the same object back in ChilloutVR.

[Source `.blend` New New new here](/Assets/src/Blender/WahahaDice.blend)

This is a typical clear dice with colored eyes on it, which schemed as such there. That's it. No explanation

We also make the Surface be like so, coz it feels perfect for Physics Gun, or whatever be thrown at in the context of Gmod-style sandboxings.

## Controls

- Use Physics Gun to drag & drop. It'll bounce sporadically that eventually stops on a surface, giving Physical Bouncing Pseudorandom effect.

## Board Game

There's board game. This is basically salvaged from our DiceGlow again.

Big shoutout to [Keianai](https://github.com/Keiannai/Prop-Game-Table-System-for-CVR) for the board game system for CVR yey.

### Change Image

- Have the Pawns here
- Hold `C`.
- Select this board
- the `wahahainsertimage` material slot changes the main game, & `wahahabottomimage` changes bottom image.
  - Choose the material that's available to you.
  - You can perhaps just plop your own `vmat` you find or upload yourself to the s&box workshop Material. 
    - Just color the `Albedo` with a square picture of board game you'd like.
- Enjoy!

You can tint the Pawn as you like. If you had a material, you can apply that too to it.

### Change Color or Material Texture

Assuming you're in `sandbox` Facepunch:

- Have the Pawns here
- Hold `C`.
- Select whichever Pawn you'd want to color
- Change `Tint` to just color it. Or change `Material` down there and choose whatever material that's available to you.
  - if unsure, `Tint`ing should suffice
- Repeat for other Pawns as necessary.
- Enjoy!
